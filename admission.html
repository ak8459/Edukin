Admissions Process


We are currently accepting applications for Grades 4 to 10 for the 2023-24 academic year.

Application Management

The procedure for admitting a student is as follows:


  1. Schools Admissions Officer Sets Admissions Window.Information Provided Here Will Be Publicly Available On The Webs information That Will BE Provided

  2. Parent/Guardian Creates An Account On The System (if He/She Does Not Already Have One) And Requests For An Application Token.

  3. System Sends A Text Message To The Parents/Guardian With The Generated Application Token.

  4. Parents/Gurdian Completes Application Form. The Application Token Can be Used To Track The Progress Of The Application From That Point.

  5. Admissions Officer is Promoted /Notified To Work On The Application After Submission

  6. Admission Officer Sets Data For Interview Or Entrance Exam(if Required)

  7. System Notifies Parents/Guardian Of The Date By Sms And Email.

  8. Admisions Officer Indicates The Admission Desion.
